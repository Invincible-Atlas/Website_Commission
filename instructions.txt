Instructions for adding a new issue to the website:
1. Go into issues.json
2. Add a new element to the list
    2A. Example for how to add a new element to the list:
        Original issues.json:
            [
                {
                    "title":"Title",
                    "body":"Body Text",
                    "image":"https://example.com/image.png",
                }


            ]
        issues.json with new element:
            [
                {
                    "title":"Title",
                    "body":"Body Text",
                    "image":"https://example.com/image.png",
                },
                {
                    "title":"Title 2",
                    "body":"Body Text 2",
                    "image":"https://example.com/image2.png",
                }


            ]   

3. Put your desired title, body text, and image link, in the proper fields.
